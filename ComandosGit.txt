Passo a passo para subir códigos para o GitHub 

COMANDOS TERMINAL LINUX

ls -> comando para mostrar em que pasta estamos
cd -> comando para entrar na pasta desejada 


COMANDOS GIT


git status -> comando para visualizar o estado do repositório local
git add . -> comando para adicionar alteracoes feitas no repositório
git status
git commit -m "mensagem descrevendo as alteracoes" -> comando para descrever o trabalho feito no repositório 
git push -> comando para upar alteracoes do repositório
