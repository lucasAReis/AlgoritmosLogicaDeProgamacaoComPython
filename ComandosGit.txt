COMANDOS TERMINAL LINUX

ls -> comando para mostrar em que pasta estamos
cd -> comando para entrar na pasta desejada 


COMANDOS GIT

git clone + Url do repositorio -> comando para fazer uma copia do repositorio no GitHub

git status -> comando para visualizar o estado do repositorio local
git add . -> comando para adicionar alteracoes feitas no repositorio
git commit -m -> "mensagem descrevendo as alteracoes" -> comando para descrever o trabalho feito no repositorio 
git push -> comando para upar alteracoes do repositorio
